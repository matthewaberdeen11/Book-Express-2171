Import Daily Sales Data

UC-001: Upload CSV exports from the ZOHO cashing system to update inventory quantities.

Supports .csv files from ZOHO export only.

CSV Formatting Guidelines

Daily Sales Import

File Format: Sales_by_Item.csv (from ZOHO export)

Required Columns:

  • item_id - Must match an existing Item ID in inventory
  • item_name - Item name (for reference)
  • quantity_sold - Number of units sold

Notes: Items not found in inventory will be marked as "unrecognized". Stock is deducted automatically. Low stock alerts generated if below threshold.

Sample CSV Format

item_id,item_name,quantity_sold
BK001,Mathematics for Grade 1,5
BK002,English Language Grade 1,3
BK003,Science Explorer Grade 2,8
BK005,Reading Comprehension Grade 2,2
BK099,Unknown Book Title,1

Note: BK099 will be flagged as unrecognised since it doesn't exist in the catalogue.
